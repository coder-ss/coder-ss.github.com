基本MVC

  • Model模型：模型的工作是按要求从数据库、接口取出数据（不全面的一个解释）
  • View视图：能只管看到的Web界面
  • Controller控制器：向系统发出指令的工具和帮手

MVC工作流程：

  1. 浏览者 -> 调用控制器，对他发出指令
  2. 控制器 -> 按指令选取一个合适的模型
  3. 模型 -> 按控制器指令取相应数据
  4. 控制器 -> 按指令选取相应视图
  5. 视图 -> 把第三步取到的数据按用户想要的样子显示处理
